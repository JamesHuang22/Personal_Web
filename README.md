## Build

1. Clone the repo:
```console

```

2. Update the *resume.json* with your personal information (check [JSON Resume](https://jsonresume.org/))

3. Install dependencies and run build command:
```console
> npm install
> npm run build
```

You can also test the app with a development server, just run:

```console
> npm start
```
